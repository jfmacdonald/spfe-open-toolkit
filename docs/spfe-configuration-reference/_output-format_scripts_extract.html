Element: extract

Location

/output-format/scripts/extract

Description

Container element for a list of extraction scripts. It is unlikely that an output type will want to contribute a script to the extraction process, and may indicate a poor separation of concerns in your architecture. However, if you do want to extract content different for a particular output format, this setting allows you to specify the appropriate extract script component.

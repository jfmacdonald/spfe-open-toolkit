Element: topic-set-groups

Location

/content-set/topic-set-groups

Description

Establishes a set of topic set groups for the content set. You can define groups and place topic sets into those groups, which forms the basis of a table of contents for the content set, or for any other way of presenting groups of topic sets. Groups can be defined hierarchically by nesting one group definition inside another. To place a topic set in a group, use the setting /content-set/topic-sets/topic-set/group.

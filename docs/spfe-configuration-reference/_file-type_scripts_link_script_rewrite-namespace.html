Element: rewrite-namespace

Location

/file-type/scripts/link/script/rewrite-namespace

Description

Instructs the SPFE build to remap the namespace declarations in an included script file. Instances of an xmlns namespace declaration or a xpath-default-namespace declaration with the namespace URI specified by from are changed to the namespace URI specified by to. A separate copy of the imported stylesheet will be included for each namespace remapping found for all topic types in a topic set. This allows you to borrow the same topic structures and scripts in several topic types and have them all processed using their correct namespaces.

Element: topic-types

Location

/topic-set/topic-types

Description

A container element for a list of topic types included in a topic set.

The list of topic types is used to locate the configuration file for each topic type, which in turn lists the scripts used to process topics of this type. The list is also used to determine the topic type order for a topic set, which determines the order in which topic types are listed in a generated table of contents.

Use

Required

Default

None

Values

N/A

Children

topic-type

Attributes

None

Restrictions

  • Every topic type occurring in the topic set must be listed here. If an unlisted topic type is included, an error will occur at the synthesis stage of the build when the synthesis scripts will be unable to find templates to process the topic type.

    Note that it is possible to write a resolve script that attempts to process an unrecognized topic type. In this case, the error described above may not occur. This is unwise, however, since such a script cannot be guaranteed to correctly interpret all the structure of an unknown type, which may result in unanticipated effects in the output.

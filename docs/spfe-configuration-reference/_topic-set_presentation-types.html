Element: presentation-types

Location

/topic-set/presentation-types

Description

A container element for a list of presentation types to be generated for a topic set. A presentation type is a way of presenting content for use in a particular class of media (online vs. paper, top-down vs. bottom-up, etc.). A presentation type is expressed as a presentation schema. Each output format is written to format a particular presentation type. Therefore, you must ensure that there is an appropriate presentation type specified for each of the output formats you specify for a topic type. The output format configuration specifies the presentation format that it uses in the setting /output-format/presentation-type. In order to build a presentation of the specified type, there must be a set of presentation scripts for this presentation type for each topic type included in the topic set. The scripts for a presentation type are contained in a /topic-type/scripts/present element with a type attribute that matches the name of the presentation type, as specified in /topic-set/presentation-types/presentation-type.

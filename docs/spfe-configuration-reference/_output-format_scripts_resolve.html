Element: resolve

Location

/output-format/scripts/resolve

Description

Container element for a list of resolve script components for an output format . The resolve scripts are responsible for assembling a set of topics in which all names are fully resolved and all conditions and inclusions have been applied. Script components contribute the handling of different elements to the resolve script.

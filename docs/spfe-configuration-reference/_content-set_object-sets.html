Element: object-sets

Location

/content-set/object-sets

Description

A container element for a list of object sets that are to be built for this content set.

Note: Object sets not included in output by default

Note that including an object set in a content set does not mean that those objects will automatically become part of the output. Objects become part of the output when they are included in topics, either through direct inclusion or via soft linking. If an object is not included by any topic, it will not be part of the output. Including different object sets include in the build is one way to manage reuse or create variant content.

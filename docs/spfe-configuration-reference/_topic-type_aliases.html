Element: aliases

Location

/topic-type/aliases

Description

A container element for a pair or topic type aliases, singular and plural. Topic types in SPFE are formally identified by the namespace URI of the topic schema. In some cases, however, we need to display the topic type to the user. For this purpose we need a user-friendly alias for the topic type. Topic type aliases provide a natural language name for the formal names of the topic types. Where you need to generate the name of a topic type (such as “API reference entry” or “Recipe”) or its plural equivalents (such as “API Reference Entry” or “API references”) from their type names ((such as “api-reference” or “recipie”), you can do so using the sf:get-topic-type-alias-singular or sf:get-topic-type-alias -plural functions respectively. Those functions will look up the alias values from this configuration setting.

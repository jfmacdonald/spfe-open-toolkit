Element: subject-types

Location

/content-set/subject-types

Description

A container for a list of subject types used in the content set. Subject types are the classification of subjects referred to by subject affinity markup and used to index topics. The subject type listing is used to provide aliases for the subject types when the name of the subject types is displayed in the output. For example, in the SPFE content, SPFE configuration settings are marked up using the config-setting element. “config-setting” is thus a subject type in this content set. However, when the output explicitly refers to a configuration setting (for instance, in a list page) the string “Configuration setting” is used in the singular and “Configuration settings” is used in the plural. These string are generated based on the value of the /content-set/subject-types/subject-type/aliases/singular and /content-set/subject-types/subject-type/aliases/plural settings for the “config-setting” subject type, as defined in the configuration file for this content set.

<subject-types>
    <subject-type>
        <id>config-setting</id>
        <aliases>
            <singular>Configuration setting</singular>
            <plural>Configuration settings</plural>
        </aliases>
    </subject-type>
...
</subject-types>

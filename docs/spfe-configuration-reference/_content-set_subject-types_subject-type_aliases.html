Element: aliases

Location

/content-set/subject-types/subject-type/aliases

Description

A container for a pair of subject type aliases. Subject type aliases provide a natural language name for the formal names of the subject types which are used to specify the type of an annotation or a topic index entry. Where you need to generate the name of a subject (such as “feature”, “API routine”, or “ingredient”) or their plural equivalents (such as “features”, “API routines”, or “ingredients”) from their type names ((such as “feature”, “api-routine”, or “ingredient”), you can do so using the sf:get-subject-type-alias-singular or sf:get-subject-type-alias-plural functions respectively. Those functions will look up the alias values from this configuration setting.

Element: output-formats

Location

/topic-set/output-formats

Description

A container for a list of output formats to be created for the topic set. You should specify an output format for each of the output formats you want the SPFE build to create for a topic set. The scripts for an output format are contained in a /topic-type/scripts/format element with a name attribute that matches the name of the format type, as specified in /topic-set/output-formats/output-format/name.

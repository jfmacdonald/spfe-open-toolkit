Element: authored-content

Location

/topic-set/sources/authored-content

Description

A container element for a list of file selectors that select the authored content files to include in the build.

Authored content includes all textual sources created by authors. It includes topics, text objects, fragments, strings (other than those defined in the configuration files), and content to be merged with extracted content.

Element: extract

Location

/structure/scripts/extract

Description

Container element for a list of extraction scripts. The extract scripts are responsible for extracting data from external sources, which may then be used to generate objects for inclusion in your content set. For example, you might use a extract script to create a chart from a set of data in a database.

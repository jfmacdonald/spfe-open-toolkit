Element: group

Location

/content-set/topic-set-groups/group/group

Description

You can define topic set groups recursively to form a hierarchy. You can define groups within groups to whatever level of depth you wish. However, publishing processes may have limits on how many level of topic set grouping they can support. It should be considered the responsibility of the publishing process to raise an error if it is presented with more levels of grouping than it expects.

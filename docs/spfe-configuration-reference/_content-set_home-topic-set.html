Element: home-topic-set

Location

/content-set/home-topic-set

Description

Designates one topic set as the home topic set for the content set. The home topic set becomes the home for the content set as a whole. In many cases, a home topic set will consist of a single topic, with the name index, which will become the home page of the content set. However, you can include as many topics as you like in the home topic set. If you only have one topic set in your content set, you should designate it as the home topic set.

Use

Required

Default

None

Values

N/A

Children

None

None

Restrictions

  • The value must match the topic set id of one topic set in the content set. Topic set IDs are defined in /topic-set/topic-set-id.

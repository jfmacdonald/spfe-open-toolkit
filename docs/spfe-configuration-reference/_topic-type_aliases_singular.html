/topic-type/aliases/singular

Description

A singular version of the topic type alias. For instance, in the SPFE documentation, the singular alias for the topic type formally identified as http://spfeopentoolkit.org/ns/spfe-docs is “Configuration setting” . The namespace alias configuration for this looks like this:

<topic-type>
    <name>{http://spfeopentoolkit.org/ns/spfe-docs}spfe-configuration-reference-entry</name>
    <aliases>
        <singular>Configuration setting</singular>
        <plural>Configuration settings</plural>
    </aliases>
    ...
</topic-type>

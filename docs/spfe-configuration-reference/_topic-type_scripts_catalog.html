Element: catalog

Location

/topic-type/scripts/catalog

Description

Container element for a list of catalog scripts. The link catalog script is responsible for creating a catalog of all the link targets in a topic set. For topics types whose resolve scripts produce a standard synthesis header with index terms embedded, the standard EPPO-simple catalog script can be used. In most cases it will be preferable to create the index header at the resolve stage rather than to produce a non-standard synthesis and then write your own link catalog script. However, that option is available and you can specify the use of your own catalog script using this setting.

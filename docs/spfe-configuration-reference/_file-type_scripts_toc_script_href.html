Element: href

Location

/file-type/scripts/toc/script/href

Description

Specifies the location of the stylesheet file to be included. You may specify either a relative or absolute local path, or a URL. If a URL is used, the SPFE build will use any XML catalogs available to it to resolve the location of the script. For the sake of portability, you should use a URL whenever possible. If it is not possible to use a URL, use a relative path wherever possible.

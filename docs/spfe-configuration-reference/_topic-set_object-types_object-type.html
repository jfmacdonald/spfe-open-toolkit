Element: object-type

Location

/topic-set/object-types/object-type

Description

An item in the list of the object types included in a topic set. The topic type is identified by its XML name (name) of the XML document that describes the individual object (for instance, a graphic record file, for a standard graphic), and the URL of its configuration file (href).

<topic-type>
    <name>{http://spfeopentoolkit.org/ns/spfe-docs}generic-topic</name>
    <href>../../topic-types/generic-topic/config.xml</href>
</topic-type>

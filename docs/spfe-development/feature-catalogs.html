Link Catalogs

Link catalogs are a feature of the build system. They are generated during the synthesis stage. They are used to generate links between topics during the presentation stage.

For topic types that use the standard EPPO-simple index markup, the topic-catalog.xsl script creates the link catalog. For topic types that do not use the standard index, you will need to write a script to generate the link catalogs for that topic type. Alternatively, you can generate index entries in the standard form in your main resolve script. These can then be processed by the standard topic-catalog.xsl script.

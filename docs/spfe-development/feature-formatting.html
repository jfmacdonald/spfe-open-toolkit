Formatting

The formatting step is the third phase of the SPFE build process. It is responsible for creating the final appearance of a topic set in a particular media.

The formatting step takes the output of a presentation process appropriate to the particular media. You might have more than one formatting process for a particular presentation. For example, you might have a different formatting process for an online PDF (color, links) and for a prepress PDF (black and white, cross references).

The formatting stage should never add anything to the content. Formatting scripts should not have to change for translation, excepting cases where a different layout is required for the target language.

Encoding

The encoding step is the final phase of the SPFE build process. It is responsible for creating a compiled deliverable, such as a PDF or EPub.

The encoding step is not needed in all cases. If the final output is an XML or HTML format, no encoding is needed as the formatting step can create the final output directly.

In most cases, the encoding step will be performed by an external tool, such as a PDF generator of the Microsoft Help Compiler. In this case, the job of the format stage it to create the correct input file or files for this tool.

Configuring the encoding stage consists of making sure that whatever configuration the external tool requires in the right place.

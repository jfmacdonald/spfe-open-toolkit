EPPO Simple Overview

EPPO Simple is a collection of document structures and associated processing code that you can use to create topic types for use with your content. EPPO Simple also includes a basic set of topic types that you can use directly if you wish. It is called EPPO Simple to indicate that it is not the only way to create schemas that conform to the Every Page is Page One (EPPO) information design principles. It merely provides a simple implementation of EPPO principles.

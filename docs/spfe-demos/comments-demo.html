Review note demo

This is a demonstration of author notes. Author notes can be included either inline in a paragraph or as a block at the text objects level.

Note: Comments not visible in final build

Because comments are intended for communication between authors and reviewers, comments are not shown in final output. So if you are looking at this page in the online docs, chances are you are not going to see any comments. To see the comments in action, do your own build of the SPFE docs with the draft build flag.
